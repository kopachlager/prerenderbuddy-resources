# JavaScript SEO Score

## What It Checks

The JavaScript SEO Score gives a practical signal about whether a JavaScript-heavy page exposes enough readable content to crawler-style requests.

## When To Use It

Use it before deciding whether to:

- Rebuild with SSR
- Use static generation
- Add prerendering
- Fix metadata or content only

## What The Results Mean

Low readable text, missing headings, missing metadata, or failed bot checks suggest the page may need a rendering fix.

The score is a technical diagnostic. It does not guarantee rankings, traffic, or AI citations.
