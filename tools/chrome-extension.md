# Prerender Buddy Chrome Extension

The free Prerender Buddy Chrome extension runs a quick crawler-readability check on the public page currently open in Chrome.

## What It Checks

- Visible content exposed by the raw HTML
- Page title and meta description
- Headings and readable page text
- JavaScript app-shell indicators
- Crawler-readability warnings

The toolbar badge shows a page score so different pages can be checked while browsing a website.

## How To Use It

1. Install the extension from the Chrome Web Store.
2. Open a public website page.
3. Select Prerender Buddy from the Chrome toolbar.
4. Run the page check and review the result.
5. Open the full web checker when a more detailed comparison is needed.

Basic checks do not require a Prerender Buddy account. Saving a site opens the Prerender Buddy web app for the normal account and setup flow.

The extension does not guarantee rankings, indexing, traffic, or AI mentions. It reports technical crawler-readability signals.
