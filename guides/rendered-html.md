# Rendered HTML

Rendered HTML is the page content after JavaScript has run and the browser has built the final document.

## Raw HTML vs Rendered HTML

Raw HTML is the first response from the server. Rendered HTML is what exists after scripts mount the app, load data, and update the page.

For client-rendered sites, these can be very different.

## Why It Matters

Search engines and AI crawlers need enough readable content to understand what a page is about. If they receive only an empty app container, metadata, and script tags, the page may be harder to interpret.

## How To Check

Use a raw-vs-rendered comparison tool and inspect whether important page content appears in the crawler response.
