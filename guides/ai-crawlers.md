# AI Crawlers

AI crawlers are automated systems that request public web pages for search, retrieval, previews, answer engines, or model-related discovery.

## Why They Matter

Search behavior is no longer only traditional web search. Public pages may be requested by Googlebot, Bingbot, GPTBot, ClaudeBot, PerplexityBot, link unfurlers, and other crawler-style systems.

Not every crawler behaves like a full browser. Some may inspect metadata and initial HTML. Others may render JavaScript partially or with limits.

## What To Make Readable

- Headings
- Body copy
- Links
- Metadata
- Canonical tags
- Open Graph tags
- Schema where relevant

## What Prerender Buddy Fixes
