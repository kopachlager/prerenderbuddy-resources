# Contributing

This repository is for public educational resources only.

## Good Contributions

- Clear examples for JavaScript-heavy public sites
- Practical troubleshooting notes
- Platform-specific setup guidance
- Plain-language explanations of crawler-readable HTML
- Corrections to broken links or outdated instructions

## Avoid

- Proprietary implementation details
- Backend, rendering, proxy, billing, auth, or infrastructure code
- Spammy SEO copy
- Promises about rankings, traffic, or AI citations
- Generic marketing language without a practical example

## Writing Style

- Use simple, direct language.
- Explain the symptom, likely cause, and next check.
- Link to the relevant Prerender Buddy tool or docs page.
- Keep examples reproducible where possible.
