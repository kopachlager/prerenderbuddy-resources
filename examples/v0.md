# v0 by Vercel

## Problem

A v0-generated page can look polished in the browser, but crawler-readable HTML still depends on how the generated code is deployed and rendered.

## Why It Happens

Vercel can host server-rendered, statically generated, and client-rendered pages. A project being hosted on Vercel does not automatically mean every route sends complete HTML from the server.

Test the generated page before deciding whether SSR, static generation, or prerendering is needed.

## How To Check It

Test the final production URL.

Check whether crawler-style HTML includes:

- Page-specific title and meta description
- H1 and visible copy
- Internal links
- Canonical URL
- Open Graph tags
- Structured data if used
